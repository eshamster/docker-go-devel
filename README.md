# Dockerfile: go-devel

A Dockerfile to configure Go development environment with Emacs.

## Installation

```bash
$ docker pull eshamster/go-devel:latest
$ docker run -v <a host folder>:/root/work -it eshamster/go-devel /bin/sh
```

## Description

This mainly consists of ...

- Go 1.16.4
- Alpine Linux 3.13
- Emacs 27.1
